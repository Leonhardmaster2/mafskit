# MafsKit

A React-based interactive math visualization library.

**Website:** [mafskit.proceduralabs.eu](https://mafskit.proceduralabs.eu)

## Structure

```
src/
├── animation/       # Animation utilities
├── context/         # React context providers
├── debug/           # Debug tools (viewport info, transform widget, FPS counter)
├── display/         # Visual components (coordinates, plots, lines, statistics)
├── gestures/        # Touch/mouse gesture handling
├── interaction/     # Interactive math elements
├── mathflow/        # Math computation engine
├── typings/         # TypeScript type definitions
└── view/            # Viewport and camera management
```

## Getting Started

```bash
pnpm install
```

## Testing

```bash
pnpm vitest
```
